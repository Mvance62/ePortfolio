I found this process very time consuming and difficult. Even though the validator would give the line number and general problem it was a guessing game to figure out exactly what the error was. I found that in some cases the validator would indicate an error on one line but I would find a problem in the line above. It usually was a missing closing tag that I forgot. There were several errors where I just needed to retype the quotation marks. A few other errors I had to correct were the order of my tr and td tags. The one thing that kept me from getting too frustrated was the fact that when I would fix one error sometimes the total number of errors would drop by more than one. This process did help me to understand the coding process a little better.
